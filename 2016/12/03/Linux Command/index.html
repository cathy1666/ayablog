Linux Command

December 3, 2016

這篇整理記錄我自己接觸 Linux 以來所用到的 Command，會一直不斷更新。

基本指令

ls -al

可list所有檔案的權限狀態

cd 路徑

更換目前工作目錄

cd ..

回上一層目錄

grep keyword filename

印出檔案中含有keyword的整行文字內容

./filename

執行(注意不用空格)(./代表現在位置)

./filename &

在背景執行(可以繼續下指令)

./start.sh 會另開shell執行start.sh，執行完結束另開的shell回到原shell
. start.sh 會在目前的shell執行start.sh

pwd

顯示目前路徑位置

ps

顯示目前正在執行的程式

ps -elf

顯示目前正在執行的程式

ps aux

顯示目前正在執行的程式

ps -T

顯示目前正在執行的thread

kill PID

終止背景執行的程式(程式的PID可用pa aux查看)

Ctrl + C

終止目前執行中的程式

init 0

關機

reboot

重開機

時間設定

date

查詢系統日期與時間

date MMDDhhmmYYYY

手動設定日期時間

hwclock -r

讀出目前 BIOS 內的時間參數

Linux 作業系統當中其實有兩個時間：

  • 軟體：Linux的系統時間，由1970/01/01開始記錄的時間參數
  • 硬體：電腦系統在 BIOS 記錄的實際時間，這也是硬體所記錄的

hwclock -w

將目前的 Linux 系統時間寫入 BIOS 當中

hwclock -w; hwclock -r; date

將軟體時鐘時間寫進硬體時鐘

更改為台灣時區

date
cp /etc/localtime /etc/localtime-old #備份原設定
cp /usr/share/zoneinfo/ROC /etc/localtime
date

台灣時區

/usr/share/zoneinfo/ROC 和
/usr/share/zoneinfo/Asia/Taipei 相同
